To implement the packet parsing state machine, we need to process the input data byte by byte. The state machine will have the following states:
1. `WAIT_START`: Wait for the START byte (0xAA).
2. `READ_LEN`: Read the length byte (LEN).
3. `READ_PAYLOAD`: Read the payload bytes based on the length.
4. `DONE`: Indicate that the processing is done.

We will use a sequential approach to maintain state across clock cycles. The state machine will transition through these states as it processes each byte of the input data.

The `data_out` will be updated based on the current state and the byte being processed. The `done` signal will be asserted when all bytes have been processed.

Here is the plan:
- Use a state register to keep track of the current state.
- Use a counter to keep track of the number of payload bytes read.
- Use a byte index to keep track of the current byte being processed.
- Update `data_out` and `done` based on the state transitions.

module dut(
    input              clk,
    input              rst,
    input              start,
    input  [63:0]      data_in,
    output reg [63:0]  data_out,
    output reg         done
);

    typedef enum logic [2:0] {
        WAIT_START,
        READ_LEN,
        READ_PAYLOAD,
        DONE
    } state_t;

    state_t state, next_state;
    reg [3:0] byte_index;
    reg [3:0] payload_count;

    always @(posedge clk or posedge rst) begin
        if (rst) begin
            state <= WAIT_START;
            byte_index <= 0;
            payload_count <= 0;
            data_out <= 64'b0;
            done <= 0;
        end else begin
            state <= next_state;
            if (next_state == WAIT_START) begin
                byte_index <= 0;
                payload_count <= 0;
                data_out <= 64'b0;
                done <= 0;

                byte_index <= byte_index + 1;
            end else if (next_state == READ_PAYLOAD) begin
                data_out[8 * (7 - byte_index) +: 8] <= data_in[8 * (7 - byte_index) +: 8];
                byte_index <= byte_index + 1;
                payload_count <= payload_count - 1;
            end else if (next_state == DONE) begin
                done <= 1;
            end
